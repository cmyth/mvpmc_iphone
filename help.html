Help

MythTV Setup

You need to provide the IP address or hostname of your master MythTV backend. You may also specify the port, if you are using a non-standard port number.

VLC Setup

In order to initiate transcoding, you will need to privide the IP address or hostname where VLC is running.

Web Server Setup

In order to watch transcoded recordings, you will need to provide the base URL for the web server that will serve the files.
